
# ChatCord-Javascript

A real time chat application using socket.io and express


## Dependencies

 - express
 - moment
 - socket.io


## Installation

Install my-project with npm

```bash
  git clone https://github.com/TarokhDev2020/ChatCord-Javascript.git
  cd my-project
  npm install
```
